Data types

    There are 8 basic data types in JavaScript.
  • number for numbers of any kind: integer or floating-point, integers are limited by ±(253-1).
  • bigint is for integer numbers of arbitrary length.
  • string for strings. A string may have zero or more characters, there’s no separate single-character type.
  • boolean for true/false.
  • null for unknown values – a standalone type that has a single value null.
  • undefined for unassigned values – a standalone type that has a single value undefined.

String Methods

  • length method is used to find the length of the string
  • indexOf method is used to find the index of word/character and returns -1 if none is found. It also takes other parameter which gives starting index to start search
  • lastIndexOf method is used to find last occurence of word or character in the string
  • search is same as indexOf
  • slice method is used to find extract substring from a given index to another
  • substring behaves same as slice
  • substr method takes index to start extraction and second as length of substring to be extracted
  • trim method removes excessive whitesapace from the string
  • charAt method finds character at a particular given index
  • toLowerCase and toUpperCase is used to convert string to upper or lower case
  • replace replaces a particular word in string with another
  • split converts string to array
  • join is used to convert array to string

Comparisons

  • null == undefined //true
  • null === undefined //false
  • OR "||" finds the first truthy value and returns it
  • AND returns the first falsy value or the last value if none were found.
